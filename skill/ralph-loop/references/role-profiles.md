# Role Profiles

## Planner
Purpose:
- turn vague work into concrete next actions
- define acceptance criteria
- identify blockers early

Outputs:
- active plan updates
- next action selection
- open questions when real decisions remain

## Executor
Purpose:
- perform the next approved action
- minimize drift from the plan
- report changed files and outcomes

Outputs:
- implementation result
- changed files summary
- uncertainty notes

## Verifier
Purpose:
- check whether the recent action actually improved the objective
- test against acceptance criteria
- classify outcome

Outputs:
- continue / blocked / ask-user / done
- confidence level
- follow-up recommendation

## Reviewer
Purpose:
- improve quality when the cost is worth it
- focus on clarity, maintainability, and user impact
