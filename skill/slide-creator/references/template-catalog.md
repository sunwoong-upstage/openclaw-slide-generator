# Template Catalog

Use this file as a decision guide when choosing slide shapes.

## Title
- Use when opening the presentation.
- Best for: title, subtitle, presenter/date context.
- Avoid using more than once.

## Heading
- Use for major section transitions.
- Best for: clear chapter breaks.
- Avoid overuse in short decks.

## Bullet
- Use for concise takeaways and structured lists.
- Best for: definitions, steps, trends, principles.
- Avoid when comparison or visual contrast matters more than listing.

## Two-column
- Use when comparing two ideas side by side.
- Best for: concept vs example, problem vs solution, before vs after.
- Avoid when one side is much heavier than the other.

## Table
- Use for compact comparison across a few dimensions.
- Best for: trend summaries, audience mappings, option comparisons.
- Avoid dense raw data dumps.

## Quote
- Use for one memorable definition, principle, or key takeaway.
- Best for: anchor sentence, closing message.
- Avoid multiple quote slides in one short deck.

## Closing
- Use for summary, recommendation, next step, or Q&A ending.
- Best for: concise end-state.
- Avoid introducing major new content here.

## Practical selection rules

- If the point can be said in one sentence, prefer `quote` or `closing`.
- If the point is a list of 3 to 5 items, prefer `bullet`.
- If the user must compare two frames, prefer `two_col`.
- If the user must compare many items across a few criteria, prefer `table`.
- If the deck is under 10 minutes, keep `heading` slides sparse.
