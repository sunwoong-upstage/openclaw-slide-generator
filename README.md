# openclaw-slide-generator

Harness Engineering 발표자료와 OpenClaw `slide-creator` 스킬을 담은 저장소입니다.

## Live

- GitHub Pages 메인: `https://sunwoong-upstage.github.io/openclaw-slide-generator/`
- 슬라이드 deck: `https://sunwoong-upstage.github.io/openclaw-slide-generator/slides/`

## 포함 내용

- `slides/` — Harness Engineering HTML 발표자료와 speaker notes
- `skill/slide-creator/` — HTML reveal.js 기반 발표자료 생성용 OpenClaw skill

## slide-creator skill 특징

- 발표 맥락 수집
- deck type 판단
- story arc planning
- slide-type selection guide
- richer presentation layouts (`metric_strip`, `card_grid`, `takeaway_band`, `feature_split`)
- HTML reveal.js 출력
- speaker notes 생성
- north-star 기반 반복 개선
- readability quality check

## 저장소 구조

- `index.html` — GitHub Pages 진입 페이지
- `slides/` — 발표 deck
- `skill/slide-creator/` — 재사용 가능한 skill source
